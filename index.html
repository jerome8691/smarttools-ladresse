📣 SmartPublish

Prépare les contenus de communication de l'agence. La publication directe Facebook/Meta sera branchée dans une étape suivante après validation.

L'image est utilisée pour prévisualiser et préparer le texte. Pour l'instant, elle n'est pas publiée automatiquement : il faudra l'ajouter manuellement dans Facebook avec le texte généré.
Pour l'instant SmartPublish prépare le texte, le script ou l'article. Si une photo est sélectionnée, elle est prévisualisée ici ; ensuite on copie/colle le texte et on ajoute l'image manuellement dans Facebook, WhatsApp ou LinkedIn.
